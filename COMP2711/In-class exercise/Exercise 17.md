## Questions

Find the least integer $n$ such that $f(x)$ is $O(x^n)$ for each of these functions.  

### $\text{Part A: }f(x) = 2x^2 + x^3 \log x$

$x = 2$

### $f(x) = 3x^5 + (\log x)^4$

$n = 5$

### $f(x) = \frac{x^4 + x^2 + 1}{x^4 + 1}$

$n = 0$

### $f(x) = \frac{x^3 + 5 \log x}{x^4 + 1}$

$x = -1$
